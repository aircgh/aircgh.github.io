A course on cloud-native geospatial engineering

Build a real Python package, one lesson at a time.

opentrash is an open-source, cloud-native geospatial Python package for residential waste collection analytics. This course builds it end to end — from raw municipal CSVs to GPS streams to an interactive map — with the engineering told straight, no copy-paste from notebooks.

The architectural heart of the package is the routing engine: a single layer that joins every GPS ping to every operational GIS layer (routes, parcels, facilities) once. Pattern detection, RouteView, and any future product is a thin consumer of engine output. Each lesson lands one piece of that architecture.
