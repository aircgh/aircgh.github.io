Open data you can browse and query—without servers.

AIRC publishes and curates open datasets with a simple model: browse via a static catalog (STAC) and query via GeoParquet + DuckDB using spatial bounds and time filters.

ISTAC (Serverless STAC + GeoParquet)

ISTAC is our immutable, versioned catalog structure for publishing STAC Collections and STAC GeoParquet “item mirrors” so users can query quickly with DuckDB/Python.

ISTAC Landing

The ISTAC directory contains the root catalog, versioned snapshots, and (soon) the browser UI.
